# Distance-Magic-TravelApp

A modern and responsive travel website built using **Tailwind CSS with a mobile-first approach**, designed to help users explore destinations, discover services, and plan their journeys effortlessly.
---
## ✨ Features
- 🌄 Explore top travel destinations  
- 🔍 Search trips by city, date, and travellers  
- 🧭 Discover services like tours, pricing, and experiences  
- 📰 Travel news section  
- 📱 Fully responsive (mobile-first design)  
- 🎨 Modern UI with glassmorphism effects  
---
## 🚀 Technologies Used
- HTML5  
- CSS3  
- Tailwind CSS (Mobile-First Design)  
- JavaScript
